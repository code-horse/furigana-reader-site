# ふりがなリーダー - ユーザーガイド

> バージョン: v1.4.0

## はじめに

ふりがなリーダーは、日本語学習者向けのブラウザ拡張機能です。WASM 形態素解析エンジン（Lindera + IPAdic）を搭載し、JavaScript にフォールバック可能なため、ウェブページや PDF 上の漢字に正確なふりがな（読みの注釈）を付けます。内蔵の日本語辞書、テキスト読み上げ、翻訳機能も搭載し、日本語の発音学習をより簡単にします。

---

## 主な機能

- **ページ全体ふりがなモード** — ワンクリックでページ内のすべての漢字にふりがなを付与
- **ホバー辞書** — 注釈付きの文字にマウスを合わせると、JMdict の定義（18万件以上）、読み、JLPT レベルバッジ（N5〜N1）、発音ボタンを表示；辞書モード、読みモード、オフから選択可能
- **PDF リーダー** — ふりがな注釈、辞書、読み上げ、翻訳機能を備えた内蔵 PDF リーダー；ドラッグ＆ドロップ、URL 読み込み、PDF スマート検出による自動リダイレクトに対応
- **3種類の表示スタイル** — ひらがな・カタカナ・ローマ字の表示に対応
- **送り仮名分割** — 送り仮名と漢字語幹を正確に分離
- **熟字訓対応** — 複数漢字の特殊読み（熟字訓）を正しく処理
- **テキスト読み上げ（TTS）** — スピーカーボタンをクリックして日本語の発音を再生
- **選択範囲の読み上げ＋カラオケ効果** — 日本語テキストを選択すると、コンパクトなツールバーが表示され、読み上げボタンと翻訳ボタンが使用可能；読み上げ中は単語または文字がリアルタイムでハイライト（カラオケ効果）
- **選択範囲の翻訳** — テキストを選択し、ツールバーの翻訳ボタンをクリックすると、Bing または Google 翻訳による即時翻訳がインラインバブルで表示
- **キーボードショートカット** — カスタマイズ可能なショートカットで主要機能にすばやくアクセス
- **多言語インターフェース** — 38 言語の UI に対応

## 使い方

### ステップ 1: 拡張機能をインストール

[Chrome ウェブストア](https://chromewebstore.google.com/)から **ふりがなリーダー** をインストールするか、開発者モードでローカル読み込みしてください。

### ステップ 2: 任意のウェブページを開く

日本語のコンテンツがあるページを開きます。

### ステップ 3: ふりがなを有効にする

ブラウザのツールバーにある拡張機能アイコンをクリックし、「ふりがなを有効にする」をオンにします。次に「ページ全体ふりがな」をオンにすると、ページ上のすべての漢字にふりがなが付きます。右下のフローティングボタンからも切り替えできます。

### ステップ 4: ふりがなを確認する

文字にマウスを合わせてふりがなのツールチップと辞書の定義を表示し、スピーカーアイコンをクリックして発音を聞きます。

### ステップ 5: 選択したテキストを読み上げ・翻訳する

マウスで任意の日本語を選択すると、選択範囲の近くにコンパクトなツールバーが表示されます：
- **🔊 読み上げ** — 選択テキストをカラオケ風ハイライト付きで読み上げ
- **🌐 翻訳** — ツールバーの下にインライン翻訳バブルを表示

右クリックして「ふりがなリーダー > 選択範囲を読み上げ」または「ふりがなリーダー > 選択範囲を翻訳」も選択できます。

> **ヒント:** ブラウザのツールバーにある拡張機能アイコンをクリックすると設定パネルが開き、ふりがなスタイル、ホバーモード、読み上げ速度、翻訳エンジンなどを調整できます。

## ホバー辞書

拡張機能には JMdict ベースの日本語辞書（18万件以上）が内蔵されています。設定でホバーモードを選択できます：

| モード | 動作 |
|--------|------|
| **辞書モード** | ホバーで読み＋定義＋JLPTレベル＋発音ボタンを表示 |
| **読みモード** | ホバーで読み＋発音ボタンを表示（定義なし） |
| **オフ** | ホバー効果なし |

**辞書モード**では、ツールチップに以下が表示されます：
- 単語とその読み（ふりがな）
- 発音ボタン（クリックで再生）
- JMdict の日本語定義
- JLPT レベルバッジ（N5〜N1、該当する場合）

> **ヒント:** 辞書データは辞書モード有効時にオンデマンドで読み込まれ、他のモードに切り替えるとメモリ節約のためアンロードされます。

## PDF リーダー

ふりがなリーダーには内蔵 PDF リーダーがあり、PDF ドキュメントでもふりがな注釈、辞書、読み上げ、翻訳機能が使えます — ウェブページで使えるすべての機能が PDF でも利用可能です。

### PDF を開く

**方法 1: ポップアップから**  
拡張機能アイコンをクリックし、「PDF リーダーを開く」をクリック。PDF ファイルをドラッグ＆ドロップするか、「ファイルを選択」をクリックしてローカル PDF を開きます。PDF の URL を貼り付けることもできます。

**方法 2: コンテキストメニュー**  
ウェブページ上の `.pdf` リンクを右クリックし、「ふりがなリーダーで PDF を開く」を選択。

**方法 3: 自動検出**  
設定で「PDF スマート検出」を有効にすると、`.pdf` URL が自動的に内蔵リーダーにリダイレクトされます。PDF が検出されたがリダイレクトされない場合（Chrome の内蔵ビューアなど）、通知とプロンプトが表示されます。

### PDF リーダーの機能

- **ふりがな注釈** — ページ全体モードやホバーツールチップなど、すべてのふりがな機能が PDF テキストで動作
- **5つのふりがなモード** — ひらがな、カタカナ、ローマ字、ホバーのみ、非表示
- **クリック辞書** — 任意の単語をクリックして JMdict の定義を表示（PDF ではホバーではなくクリックを使用し、集中した読書体験を提供）
- **選択ツールバー** — テキストを選択して読み上げ、翻訳、コピー
- **サイドバー** — 目次アウトラインとページサムネイル
- **検索** — PDF 内の全文検索
- **テーマ** — ダーク、ライト、セピアの3つの読書テーマ
- **ズーム** — ズーム対応ふりがなを含む複数のズームレベル
- **キーボードショートカット** — 矢印キーでナビゲーション、+/- でズーム、Ctrl/Cmd+F で検索

## 選択範囲の読み上げとカラオケ

選択範囲の読み上げ機能では、任意の日本語テキストを選択してワンクリックで読み上げ — 文の発音学習や読解練習に最適です。

**方法 1: 選択ツールバー**  
マウスで日本語テキストを選択すると、選択範囲の近くにコンパクトなツールバーが表示されます（🔊読み上げボタンと🌐翻訳ボタン）。読み上げボタンをクリックして再生すると、各単語または文字がリアルタイムでハイライト（カラオケ効果）されます。

**方法 2: 右クリックメニュー**  
日本語テキストを選択後、右クリックして「ふりがなリーダー > 選択範囲を読み上げ」を選択。

**方法 3: キーボードショートカット**  
テキストを選択して `Alt+Shift+S`（Mac: `Ctrl+Shift+S`）を押して読み上げ。

> **ヒント:** カラオケハイライト効果は、ブラウザが TTS ワード境界イベントをサポートしている場合に最適です。サポートされていない場合、拡張機能はタイミングベースのフォールバックを使用してスムーズなハイライトを実現します。

## 翻訳

ページ上の任意のテキストを選択して、翻訳機能で即時翻訳を取得できます。

**方法 1: 選択ツールバー**  
テキストを選択し、ツールバーの🌐翻訳ボタンをクリック。翻訳結果がバブルで表示され、コピーボタン付き。

**方法 2: 右クリックメニュー**  
テキストを選択し、右クリックして「ふりがなリーダー > 選択範囲を翻訳」を選択。

**方法 3: キーボードショートカット**  
テキストを選択して `Alt+Shift+T`（Mac: `Ctrl+Shift+T`）を押して翻訳。

**翻訳エンジン:**
- **Bing 翻訳**（デフォルト）— Microsoft Translator 提供
- **Google 翻訳** — Google 提供

両エンジンとも **108 の対象言語** に対応。

拡張機能の設定で翻訳エンジンと対象言語を切り替えられます。対象言語はブラウザの言語から自動検出されます。

> **ヒント:** ツールバーやバブルの外側をクリックすると閉じます。

## キーボードショートカット

| ショートカット | Mac ショートカット | アクション |
|---------------|-------------------|-----------|
| `Alt+Shift+F` | `Ctrl+Shift+F` | ふりがな注釈のオン／オフ切り替え |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 選択テキストを読み上げ |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 選択テキストを翻訳 |

> **ヒント:** Chrome の `chrome://extensions/shortcuts` でショートカットをカスタマイズできます。

## 設定ガイド

| 設定 | 説明 |
|---------|-------------|
| **ふりがなを有効にする** | ふりがな機能全体のオン／オフ |
| **ページ全体ふりがな** | 有効にするとすべての漢字にふりがなを表示（レイアウトに影響する場合があります） |
| **ホバーモード** | ホバー動作を選択：辞書モード（読み＋定義＋JLPT＋音声）、読みモード（読み＋音声）、またはオフ |
| **ふりがなスタイル** | ひらがな・カタカナ・ローマ字から選択 |
| **読み上げ速度** | 文の読み上げの速さを調整 |
| **翻訳エンジン** | Bing 翻訳または Google 翻訳を選択 |
| **対象言語** | 翻訳の対象言語を設定（ブラウザの言語から自動検出） |
| **PDF スマート検出** | 有効にすると PDF URL を自動的に内蔵リーダーにリダイレクトし、PDF 検出時に通知を表示 |

## よくある質問

**Q: 一部のページでは動作しません。**  
A: セキュリティ上の理由により、`chrome://` やブラウザ設定、Chrome ウェブストアなどの特殊なページでは拡張機能は動作しません。

**Q: ふりがなが不正確な場合は？**  
A: まれな語や特殊な読み（熟字訓など）では誤りが出ることがあります。改善を続けていますので、具体的な事例を共有いただけると助かります。

**Q: 読み上げで音が出ません。**  
A: システムの音量設定を確認し、日本語の音声パックがインストールされているか確認してください。ブラウザや OS によって読み上げの対応は異なります。

**Q: ページ全体モードでレイアウトが崩れます。**  
A: ふりがなには余分なスペースが必要なため、元のページレイアウトに影響することがあります。読みにくい場合はページ全体モードをオフにし、ホバーのツールチップをご利用ください。

**Q: 翻訳が動作しません。**  
A: 翻訳にはインターネット接続が必要です。Bing 翻訳が失敗する場合は、設定で Google 翻訳に切り替えてみてください。一部のネットワーク環境では翻訳サービスへのアクセスがブロックされることがあります。

**Q: ふりがなリーダーで PDF を開くには？**  
A: いくつかの方法があります：ポップアップで「PDF リーダーを開く」をクリック、PDF リンクを右クリックして「ふりがなリーダーで PDF を開く」を選択、または設定で「PDF スマート検出」を有効にして PDF URL を自動リダイレクト。

**Q: PDF スマート検出を有効にしても一部の PDF がリダイレクトされません。**  
A: 自動リダイレクトは `.pdf` で終わる URL に対して機能します。`.pdf` 拡張子のない PDF や Chrome の内蔵ビューアで表示される PDF の場合、通知とバッジが表示されます。

**Q: 辞書はオフラインで使えますか？**  
A: はい。JMdict 辞書（18万件以上）は拡張機能に完全に内蔵されています。すべての検索はネットワーク接続なしでローカルで実行されます。
