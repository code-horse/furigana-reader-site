# Furigana Reader - ユーザーガイド

> バージョン: v2.0.0

## はじめに

Furigana Reader は、日本語学習者向けのブラウザ拡張機能です。WASM 形態素解析エンジン（Lindera）を搭載し、JavaScript にフォールバック可能なため、ウェブページ上の漢字に正確なふりがな（読みの注釈）を付け、日本語の発音学習をより簡単にします。

---

## 主な機能

- **テキスト選択による注釈** — ページ上の日本語テキストを選択すると、ふりがなと読み上げボタンを自動表示
- **ページ全体ふりがなモード** — ワンクリックでページ内のすべての漢字にふりがなを付与
- **テキスト読み上げ（TTS）** — スピーカーボタンをクリックして日本語の発音を再生
- **選択範囲の読み上げ** — 日本語テキストを選択し、フローティングボタンをクリックするか、右クリックで「選択範囲を読み上げ」を選ぶ
- **ホバーによるツールチップ** — 注釈付きの文字にマウスを合わせてふりがなと発音ボタンを表示
- **複数のふりがなスタイル** — ひらがな・カタカナ・ローマ字の表示に対応
- **多言語インターフェース** — 38 言語の UI に対応

## 使い方

### ステップ 1: 拡張機能をインストール

[Chrome ウェブストア](https://chromewebstore.google.com/)から **Furigana Reader** をインストールするか、開発者モードでローカル読み込みしてください。

### ステップ 2: 任意のウェブページを開く

日本語のコンテンツがあるページを開きます。

### ステップ 3: テキストを選択するかフローティングボタンを使う

注釈を付けたい日本語を選択するか、右下のフローティングボタンをクリックしてページ全体ふりがなモードを有効にします。

### ステップ 4: ふりがなを確認する

文字にマウスを合わせてふりがなのツールチップを表示し、スピーカーアイコンをクリックして発音を聞きます。

### ステップ 5: 選択したテキストを読み上げる

マウスで任意の日本語を選択し、フローティングのスピーカーボタンをクリックするか、右クリックして「選択範囲を読み上げ」を選びます。

> **ヒント:** ブラウザのツールバーにある拡張機能アイコンをクリックすると設定パネルが開き、ふりがなスタイルや読み上げ速度などを調整できます。

## 設定ガイド

| 設定 | 説明 |
|---------|-------------|
| **ふりがなを有効にする** | ふりがな機能全体のオン／オフ |
| **ページ全体ふりがな** | 有効にするとすべての漢字にふりがなを表示（レイアウトに影響する場合があります） |
| **ふりがなスタイル** | ひらがな・カタカナ・ローマ字から選択 |
| **読み上げ速度** | 文の読み上げの速さを調整 |
| **ホバー時のツールチップ** | マウスホバーでふりがなのツールチップを表示 |

## よくある質問

**Q: 一部のページでは動作しません。**  
A: セキュリティ上の理由により、`chrome://` やブラウザ設定、Chrome ウェブストアなどの特殊なページでは拡張機能は動作しません。

**Q: ふりがなが不正確な場合は？**  
A: まれな語や特殊な読み（熟字訓など）では誤りが出ることがあります。改善を続けていますので、具体的な事例を共有いただけると助かります。

**Q: 読み上げで音が出ません。**  
A: システムの音量設定を確認し、日本語の音声パックがインストールされているか確認してください。ブラウザや OS によって読み上げの対応は異なります。

**Q: ページ全体モードでレイアウトが崩れます。**  
A: ふりがなには余分なスペースが必要なため、元のページレイアウトに影響することがあります。読みにくい場合はページ全体モードをオフにし、ホバーのツールチップをご利用ください。
