# Furigana Reader - 사용자 안내

> 버전: v1.3.0

## 소개

Furigana Reader는 일본어 학습자를 위한 브라우저 확장 프로그램입니다. WASM 형태소 분석기(Lindera)를 사용하며 필요 시 JavaScript로 대체되어, 웹 페이지의 한자에 후리가나(읽기 주석)를 정확히 달아 일본어 발음 학습을 더 쉽게 도와줍니다.

---

## 주요 기능

- **텍스트 선택 주석** — 웹 페이지에서 일본어 텍스트를 선택하면 후리가나와 음성 버튼이 자동으로 표시됩니다
- **전체 페이지 후리가나 모드** — 한 번의 클릭으로 페이지의 모든 한자에 후리가나를 표시합니다
- **텍스트 음성 변환(TTS)** — 스피커 버튼을 눌러 일본어 발음을 들을 수 있습니다
- **선택 영역 읽기** — 일본어 텍스트를 선택한 뒤 플로팅 버튼을 누르거나, 오른쪽 클릭으로 “선택 영역 읽기”를 선택합니다
- **호버 툴팁** — 주석이 달린 글자에 마우스를 올리면 후리가나와 발음 버튼이 표시됩니다
- **여러 후리가나 스타일** — 히라가나, 가타카나, 로마자 표시를 지원합니다
- **다국어 인터페이스** — 38개 인터페이스 언어를 지원합니다

---

## 사용 방법

### 1단계: 확장 프로그램 설치

[Chrome 웹스토어](https://chromewebstore.google.com/)에서 **Furigana Reader**를 설치하거나, 개발자 모드로 로컬에서 불러오세요.

### 2단계: 웹 페이지 열기

일본어 콘텐츠가 있는 아무 웹 페이지나 방문합니다.

### 3단계: 텍스트 선택 또는 플로팅 버튼 사용

주석을 달고 싶은 일본어를 선택하거나, 오른쪽 아래 플로팅 버튼을 눌러 전체 페이지 후리가나 모드를 켭니다.

### 4단계: 후리가나 확인

글자에 마우스를 올려 툴팁을 보고, 스피커 아이콘을 눌러 발음을 들습니다.

### 5단계: 선택한 텍스트 읽기

마우스로 일본어를 선택한 뒤 플로팅 스피커 버튼을 누르거나, 오른쪽 클릭으로 “선택 영역 읽기”를 선택합니다.

> **팁:** 브라우저 도구 모음의 확장 프로그램 아이콘을 누르면 설정 패널이 열리며, 후리가나 스타일·음성 속도 등을 조정할 수 있습니다.

## 설정 안내

| 설정 | 설명 |
|---------|-------------|
| **후리가나 사용** | 후리가나 기능 전체를 켜거나 끕니다 |
| **전체 페이지 후리가나** | 켜면 모든 한자에 후리가나를 표시합니다(레이아웃에 영향을 줄 수 있음) |
| **후리가나 스타일** | 히라가나, 가타카나 또는 로마자 중 선택 |
| **음성 속도** | 문장 읽기 속도를 조절합니다 |
| **호버 툴팁** | 마우스를 올리면 후리가나 툴팁을 표시합니다 |

## 자주 묻는 질문

**Q: 일부 페이지에서는 작동하지 않아요.**  
A: 보안상의 이유로 `chrome://`, 브라우저 설정, Chrome 웹스토어 같은 특수 페이지에서는 확장 프로그램이 실행되지 않습니다.

**Q: 후리가나가 부정확하면요?**  
A: 드문 단어나 특수 읽기(숙자훈 등)에서 오류가 날 수 있습니다. 지속적으로 개선 중이며, 구체적인 사례를 알려주시면 도움이 됩니다.

**Q: 음성 변환에서 소리가 나지 않아요.**  
A: 시스템 볼륨과 일본어 음성 팩 설치 여부를 확인하세요. 브라우저와 운영체제에 따라 음성 지원이 다릅니다.

**Q: 전체 페이지 모드가 레이아웃을 망가뜨려요.**  
A: 후리가나에는 여유 공간이 필요해 원래 페이지 레이아웃에 영향을 줄 수 있습니다. 읽기 불편하면 전체 페이지 모드를 끄고 호버 툴팁을 사용하세요.
