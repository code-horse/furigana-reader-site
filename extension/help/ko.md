# Furigana Reader - 사용자 안내

> 버전: v1.4.3

## 소개

Furigana Reader는 일본어 학습자를 위한 브라우저 확장 프로그램입니다. WASM 형태소 분석기(Lindera + IPAdic)와 JavaScript 폴백으로 동작하며, 웹 페이지와 PDF의 한자에 후리가나(읽기 주석)를 정확히 붙입니다. 내장 일본어 사전, 텍스트 음성 변환(TTS), 번역 기능도 포함되어 있어 일본어 발음 학습을 더 쉽게 할 수 있습니다.

---

## 주요 기능

- **전체 페이지 후리가나 모드** — 한 번의 클릭으로 페이지의 모든 한자에 후리가나 주석 추가
- **호버 사전** — 주석이 붙은 글자에 마우스를 올리면 JMdict 정의(18만 개 이상 항목), 읽기, JLPT 레벨 배지(N5–N1), 발음 버튼 표시; 사전 모드, 읽기 모드, 끄기 중 선택
- **PDF 리더** — 후리가나, 사전, 음성, 번역이 있는 내장 PDF 리더; 드래그 앤 드롭, URL 로드, 자동 PDF 감지 및 스마트 리다이렉트 지원
- **세 가지 읽기 스타일** — 히라가나, 가타카나, 로마자 표시
- **오쿠리가나 분리** — 오쿠리가나(送り仮名)를 한자 어간에서 정확히 분리
- **쥬쿠지쿤 지원** — 특수 읽기를 가진 다자 한자 복합어(熟字訓)의 올바른 읽기
- **텍스트 음성 변환** — 스피커 버튼을 눌러 일본어 발음 듣기
- **선택 영역 음성 + 카라오케 효과** — 일본어 텍스트를 선택하면 작은 도구 모음이 나타나 음성·번역 버튼 제공; 재생 시 단어별 또는 글자별 실시간 강조(카라오케 효과)가 오디오와 동기화
- **선택 영역 번역** — 텍스트를 선택한 뒤 도구 모음의 번역 버튼을 누르면 Bing 또는 Google 번역으로 즉시 번역, 인라인 말풍선으로 표시
- **키보드 단축키** — 사용자 지정 가능한 단축키로 핵심 기능에 빠르게 접근
- **다국어 인터페이스** — 38개 인터페이스 언어 지원

## 사용 방법

### 1단계: 확장 프로그램 설치
[Chrome 웹 스토어](https://chromewebstore.google.com/)에서 **Furigana Reader**를 설치하거나, 개발자 모드로 로컬에서 불러옵니다.

### 2단계: 웹 페이지 열기
일본어 콘텐츠가 있는 아무 웹 페이지나 방문합니다.

### 3단계: 후리가나 켜기
브라우저 도구 모음의 확장 프로그램 아이콘을 클릭합니다. 「후리가나 사용」을 켠 다음 「전체 페이지 후리가나」를 켜 페이지의 모든 한자에 주석을 붙입니다. 오른쪽 하단의 플로팅 버튼을 눌러도 됩니다.

### 4단계: 후리가나 보기
글자에 마우스를 올리면 읽기와 사전 정의가 있는 후리가나 툴팁이 표시됩니다. 스피커 아이콘을 눌러 발음을 들을 수 있습니다.

### 5단계: 선택한 텍스트 읽기 및 번역
마우스로 일본어 텍스트를 선택합니다. 선택 영역 근처에 작은 도구 모음이 나타나며 두 개의 버튼이 있습니다:
- **🔊 읽기** — 선택한 텍스트를 카라오케 스타일 강조와 함께 읽어 줍니다
- **🌐 번역** — 도구 모음 아래에 인라인 번역 말풍선을 표시합니다

또한 오른쪽 클릭 후 「Furigana Reader > 선택 영역 읽기」 또는 「Furigana Reader > 선택 영역 번역」을 선택할 수 있습니다.

> **팁:** 브라우저 도구 모음의 확장 프로그램 아이콘을 클릭해 설정 패널을 열고 후리가나 스타일, 호버 모드, 음성 속도, 번역 엔진 등을 조정할 수 있습니다.

## 호버 사전

확장 프로그램에는 JMdict(18만 개 이상 항목) 기반의 내장 일본어 사전이 있습니다. 설정에서 여러 호버 모드를 선택할 수 있습니다:

| 모드 | 동작 |
|------|------|
| **사전** | 호버 시 읽기 + 정의 + JLPT 레벨 + 발음 버튼 표시 |
| **읽기** | 호버 시 읽기 + 발음 버튼만 표시(정의 없음) |
| **끄기** | 호버 효과 없음 |

**사전 모드**에서는 툴팁에 다음이 표시됩니다:
- 단어와 읽기(후리가나)
- 발음 버튼(클릭하여 재생)
- JMdict의 일본어 정의
- 있는 경우 JLPT 레벨 배지(N5–N1)

> **팁:** 사전 데이터는 사전 모드가 켜져 있을 때 필요 시 로드되며, 다른 모드로 바꾸면 메모리 절약을 위해 언로드됩니다.

## PDF Reader

Furigana Reader에는 내장 PDF 리더가 있어, 웹 페이지에서 쓰던 후리가나·사전·음성·번역 기능을 PDF에서도 그대로 이용할 수 있습니다.

### PDF 열기

**방법 1: 팝업에서**  
확장 프로그램 아이콘을 클릭한 뒤 「PDF 리더 열기」를 누릅니다. PDF 파일을 드래그 앤 드롭하거나 「파일 선택」으로 로컬 PDF를 엽니다. PDF URL을 붙여넣을 수도 있습니다.

**방법 2: 컨텍스트 메뉴**  
웹 페이지의 `.pdf` 링크를 오른쪽 클릭하고 「Furigana Reader로 PDF 열기」를 선택합니다.

**방법 3: 자동 감지**  
설정에서 「PDF 스마트 감지」를 켜 두면 확장 프로그램이 `.pdf` URL을 내장 리더로 자동 리다이렉트합니다. PDF가 감지되었지만 리다이렉트되지 않은 경우(예: Chrome 기본 뷰어) 알림과 안내가 표시되어 Furigana Reader에서 열도록 할 수 있습니다.

### PDF 리더 기능

- **후리가나 주석** — 전체 페이지 모드·호버 툴팁 등 모든 후리가나 기능이 PDF 텍스트에서 동작합니다
- **다섯 가지 후리가나 모드** — 히라가나, 가타카나, 로마자, 호버만, 끄기
- **클릭 사전** — 단어를 클릭하면 JMdict 정의 표시(PDF에서는 방해 없는 읽기를 위해 호버 대신 클릭 사용)
- **선택 도구 모음** — 텍스트를 선택해 읽기, 번역, 복사
- **사이드바** — 목차 개요 및 페이지 썸네일
- **검색** — PDF 전체 텍스트 검색
- **테마** — 다크, 라이트, 세피아 읽기 테마
- **확대/축소** — 여러 배율 및 확대에 맞춘 후리가나
- **키보드 단축키** — 화살표 키로 탐색, +/- 로 확대/축소, Ctrl/Cmd+F 로 검색

## 선택 영역 음성 및 카라오케

선택 영역 음성 기능으로 일본어 텍스트를 선택한 뒤 한 번의 클릭으로 읽어 줄 수 있어 문장 발음 학습과 읽기 연습에 적합합니다.

**방법 1: 선택 도구 모음**  
마우스로 일본어 텍스트를 선택합니다. 선택 근처에 🔊 읽기 버튼과 🌐 번역 버튼이 있는 작은 도구 모음이 나타납니다. 읽기 버튼을 눌러 재생합니다. 읽는 동안 각 단어나 글자가 실시간으로 강조되어(카라오케 효과) 따라 읽기 쉽습니다.

**방법 2: 오른쪽 클릭 메뉴**  
일본어 텍스트를 선택한 뒤 오른쪽 클릭하고 「Furigana Reader > 선택 영역 읽기」를 선택합니다.

**방법 3: 키보드 단축키**  
텍스트를 선택한 뒤 `Alt+Shift+S`(Mac: `Ctrl+Shift+S`)를 누릅니다.

> **팁:** 카라오케 강조는 브라우저가 TTS 단어 경계 이벤트를 지원할 때 가장 잘 동작합니다. 지원되지 않으면 확장 프로그램이 시간 기반 폴백으로 부드럽게 강조합니다.

## 번역

페이지에서 텍스트를 선택한 뒤 번역 기능으로 즉시 번역을 받을 수 있습니다.

**방법 1: 선택 도구 모음**  
텍스트를 선택한 뒤 도구 모음의 🌐 번역 버튼을 클릭합니다. 아래에 번역 말풍선이 나타나 결과와 복사 버튼이 표시됩니다.

**방법 2: 오른쪽 클릭 메뉴**  
텍스트를 선택하고 오른쪽 클릭 후 「Furigana Reader > 선택 영역 번역」을 선택합니다.

**방법 3: 키보드 단축키**  
텍스트를 선택한 뒤 `Alt+Shift+T`(Mac: `Ctrl+Shift+T`)를 누릅니다.

**번역 엔진:**
- **Bing 번역** (기본) — Microsoft Translator 기반
- **Google 번역** — Google 기반

두 엔진 모두 **108개 목표 언어**를 지원합니다.

확장 프로그램 설정에서 번역 엔진과 목표 언어를 바꿀 수 있습니다. 목표 언어는 브라우저 언어에서 자동으로 감지됩니다.

> **팁:** 도구 모음이나 말풍선 바깥 아무 곳이나 클릭하면 닫힙니다.

## 키보드 단축키

| 단축키 | Mac 단축키 | 동작 |
|--------|------------|------|
| `Alt+Shift+F` | `Ctrl+Shift+F` | 후리가나 주석 켜기/끄기 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 선택한 텍스트 읽기 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 선택한 텍스트 번역 |

> **팁:** Chrome에서 `chrome://extensions/shortcuts`로 이동해 이 단축키를 사용자 지정할 수 있습니다.

## FAQ

**Q: 일부 페이지에서 동작하지 않는 이유는?**  
A: 보안상의 이유로 확장 프로그램은 `chrome://`, 브라우저 설정, Chrome 웹 스토어 같은 특수 페이지에서는 실행할 수 없습니다.

**Q: 후리가나가 부정확하면?**  
A: 드문 단어나 특수 읽기(熟字訓)에서 오류가 있을 수 있습니다. 지속적으로 개선 중이며, 구체적인 사례를 알려 주시면 개선에 도움이 됩니다.

**Q: TTS에 소리가 안 나요?**  
A: 시스템 볼륨과 일본어 음성 팩 설치 여부를 확인하세요. 음성 지원은 브라우저와 OS마다 다릅니다.

**Q: 전체 페이지 모드가 레이아웃을 망가뜨리나요?**  
A: 후리가나 주석은 추가 공간이 필요해 원래 페이지 레이아웃에 영향을 줄 수 있습니다. 읽기에 방해되면 전체 페이지 모드를 끄고 호버 툴팁만 사용하세요.

**Q: 번역이 안 돼요?**  
A: 번역에는 인터넷 연결이 필요합니다. Bing 번역이 실패하면 설정에서 Google 번역으로 바꿔 보세요. 일부 네트워크에서는 번역 서비스 접근이 차단될 수 있습니다.

**Q: Furigana Reader로 PDF를 어떻게 열나요?**  
A: 팝업에서 「PDF 리더 열기」를 클릭하거나, PDF 링크를 오른쪽 클릭해 「Furigana Reader로 PDF 열기」를 선택하거나, 설정에서 「PDF 스마트 감지」를 켜 `.pdf` URL을 내장 리더로 자동 리다이렉트할 수 있습니다.

**Q: PDF 스마트 감지를 켰는데 일부 PDF는 리다이렉트되지 않아요?**  
A: 자동 리다이렉트는 URL이 `.pdf`로 끝나는 경우에 동작합니다. 확장자 없이 제공되는 PDF나 Chrome 기본 뷰어로 연 경우 알림과 배지로 Furigana Reader에서 열도록 안내됩니다.

**Q: 사전을 오프라인으로 쓸 수 있나요?**  
A: 네. JMdict 사전(18만 개 이상 항목)은 확장 프로그램에 모두 포함되어 있으며, 모든 조회는 네트워크 없이 로컬에서 수행됩니다.
