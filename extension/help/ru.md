# Furigana Reader — Руководство пользователя

> Версия: v1.4.1

## Введение

Furigana Reader — расширение для браузера для изучающих японский язык. На основе WASM-морфологического анализатора (Lindera + IPAdic) с резервным вариантом на JavaScript оно точно добавляет к kanji на веб-страницах и в PDF фуригана (подсказки чтения). Встроены японский словарь, синтез речи и перевод — чтобы проще осваивать произношение.

---

## Основные возможности

- **Режим фуригана для всей страницы** — Одним щелчком добавьте фуригана ко всем kanji на странице
- **Словарь при наведении** — Наведите курсор на аннотированные символы, чтобы увидеть значения из JMdict (более 180 000 записей), чтения, значки уровня JLPT (N5–N1) и кнопки произношения; выберите режим Словарь, Чтение или Выкл.
- **Чтение PDF** — Встроенный просмотр PDF с фуригана, словарём, озвучкой и переводом; поддержка перетаскивания файла, открытия по URL и автоматическое обнаружение PDF с умным перенаправлением
- **Три стиля отображения** — Хирагана, катакана и ромадзи
- **Разделение okurigana** — Точно отделяет okurigana (送り仮名) от основы в kanji
- **Поддержка jukujikun** — Корректные чтения для сложных сочетаний kanji со специальным чтением (熟字訓)
- **Синтез речи** — Нажмите кнопку динамика, чтобы услышать японское произношение
- **Озвучивание выделения с караоке** — Выделите любой японский текст; появляется компактная панель с озвучкой и переводом; при воспроизведении слова или символы подсвечиваются в реальном времени (эффект караоке) в такт аудио
- **Перевод выделения** — Выделите текст, нажмите «перевести» на панели — мгновенный перевод через Bing или Google Translate во всплывающей панели под текстом
- **Сочетания клавиш** — Быстрый доступ к основным функциям через настраиваемые горячие клавиши
- **Многоязычный интерфейс** — 38 языков интерфейса

## Как пользоваться

### Шаг 1. Установите расширение

Установите **Furigana Reader** из [Chrome Web Store](https://chromewebstore.google.com/) или загрузите локально в режиме разработчика.

### Шаг 2. Откройте любую страницу

Откройте страницу с японским текстом.

### Шаг 3. Включите фуригана

Нажмите значок расширения на панели инструментов. Включите «Включить фуригана», затем «Фуригана на всей странице», чтобы аннотировать все kanji. Можно также использовать плавающую кнопку внизу справа.

### Шаг 4. Просмотрите фуригана

Наведите курсор на символы, чтобы увидеть подсказки с чтениями и значениями из словаря. Нажмите значок динамика для прослушивания.

### Шаг 5. Озвучьте и переведите выделенный текст

Выделите японский текст мышью. Рядом с выделением появляется компактная панель с двумя кнопками:
- **🔊 Озвучить** — читает выделенный текст вслух с подсветкой в стиле караоке
- **🌐 Перевести** — показывает встроенную панель перевода под панелью инструментов

Также можно щёлкнуть правой кнопкой и выбрать «Furigana Reader > Прочитать выделенное» или «Furigana Reader > Перевести выделенное».

> **Совет:** Нажмите значок расширения, чтобы открыть настройки: стиль фуригана, режим при наведении, скорость речи, движок перевода и другое.

## Словарь при наведении

В расширении встроен японский словарь на базе JMdict (более 180 000 записей). В настройках можно выбрать режим при наведении:

| Режим | Поведение |
|-------|-----------|
| **Словарь** | При наведении: чтение + значение + уровень JLPT + кнопка произношения |
| **Чтение** | При наведении: чтение + кнопка произношения (без значений) |
| **Выкл.** | Нет реакции при наведении |

В режиме **Словарь** подсказка показывает:
- Слово и его чтение (фуригана)
- Кнопку произношения (щелчок для прослушивания)
- Толкования на японском из JMdict
- Значок уровня JLPT (N5–N1), если есть

> **Совет:** Данные словаря подгружаются по запросу в режиме Словарь и выгружаются при переключении на другие режимы для экономии памяти.

## Чтение PDF

В Furigana Reader есть встроенный просмотр PDF: фуригана, словарь, озвучка и перевод — те же возможности, что и на веб-страницах, теперь для PDF.

### Как открыть PDF

**Способ 1: из всплывающего окна**  
Нажмите значок расширения, затем «Открыть чтение PDF». Перетащите файл PDF или нажмите «Выбрать файл» для локального PDF. Можно вставить URL PDF.

**Способ 2: контекстное меню**  
Щёлкните правой кнопкой по ссылке `.pdf` и выберите «Открыть PDF в Furigana Reader».

**Способ 3: автоматическое обнаружение**  
При включённой в настройках «Умной детекции PDF» расширение автоматически перенаправляет URL с `.pdf` во встроенный просмотрщик. Если PDF обнаружен, но перенаправления нет (например, встроенный просмотр Chrome), появятся уведомления и предложение открыть в Furigana Reader.

### Возможности просмотра PDF

- **Фуригана** — все функции фуригана для текста PDF, включая режим всей страницы и подсказки при наведении
- **Пять режимов фуригана** — Хирагана, катакана, ромадзи, только при наведении и выкл.
- **Словарь по щелчку** — Щёлкните по слову, чтобы увидеть значение в JMdict (в PDF используется щелчок вместо наведения для спокойного чтения)
- **Панель выделения** — Выделите текст, чтобы озвучить, перевести или скопировать
- **Боковая панель** — Оглавление и миниатюры страниц
- **Поиск** — Полнотекстовый поиск по PDF
- **Темы** — Тёмная, светлая и сепия
- **Масштаб** — Несколько уровней масштаба, в том числе фуригана с учётом масштаба
- **Сочетания клавиш** — Стрелки для навигации, +/- для масштаба, Ctrl/Cmd+F для поиска

## Озвучивание выделения и караоке

Озвучивание выделения позволяет выделить любой японский текст и прослушать его одним щелчком — удобно для произношения фраз и тренировки чтения.

**Способ 1: панель выделения**  
Выделите японский текст мышью. Появляется компактная панель с 🔊 озвучить и 🌐 перевести. Нажмите озвучить для воспроизведения. Во время чтения каждое слово или символ подсвечиваются в реальном времени (эффект караоке).

**Способ 2: контекстное меню**  
После выделения японского текста щёлкните правой кнопкой и выберите «Furigana Reader > Прочитать выделенное».

**Способ 3: сочетание клавиш**  
Выделите текст и нажмите `Alt+Shift+S` (Mac: `Ctrl+Shift+S`) для озвучивания.

> **Совет:** Подсветка караоке лучше работает, если браузер поддерживает события границ слов TTS. Иначе расширение использует запасной вариант по времени.

## Перевод

Выделите любой текст на странице и воспользуйтесь переводом для мгновенного результата.

**Способ 1: панель выделения**  
Выделите текст и нажмите 🌐 перевести на панели. Ниже появляется панель с результатом и кнопкой копирования.

**Способ 2: контекстное меню**  
Выделите текст, щёлкните правой кнопкой и «Furigana Reader > Перевести выделенное».

**Способ 3: сочетание клавиш**  
Выделите текст и нажмите `Alt+Shift+T` (Mac: `Ctrl+Shift+T`) для перевода.

**Движки перевода:**
- **Bing Translate** (по умолчанию) — Microsoft Translator
- **Google Translate** — Google

Оба поддерживают **108 целевых языков**.

Движок и целевой язык можно сменить в настройках расширения. Целевой язык по умолчанию определяется по языку браузера.

> **Совет:** Щёлкните вне панели или пузыря перевода, чтобы закрыть их.

## Сочетания клавиш

| Сочетание | Mac | Действие |
|-----------|-----|----------|
| `Alt+Shift+F` | `Ctrl+Shift+F` | Включить/выключить фуригана |
| `Alt+Shift+S` | `Ctrl+Shift+S` | Озвучить выделенный текст |
| `Alt+Shift+T` | `Ctrl+Shift+T` | Перевести выделенный текст |

> **Совет:** Назначить свои сочетания можно в Chrome на странице `chrome://extensions/shortcuts`.

## Частые вопросы

**В: Почему на некоторых страницах не работает?**  
О: Из соображений безопасности расширения не работают на особых страницах (`chrome://`), в настройках браузера и в Chrome Web Store.

**В: Что делать, если фуригана неточны?**  
О: Редкие слова или особые чтения (jukujikun) могут отображаться с ошибками. Мы постоянно улучшаем движок; конкретные примеры нам помогают.

**В: Нет звука при синтезе речи?**  
О: Проверьте громкость системы и наличие японских голосов. Поддержка зависит от браузера и ОС.

**В: Режим всей страницы портит вёрстку?**  
О: Фуригана занимают дополнительное место. Если мешает чтению, отключите режим всей страницы и используйте подсказки при наведении.

**В: Перевод не работает?**  
О: Нужен доступ в интернет. Если Bing Translate не отвечает, попробуйте Google Translate в настройках. В некоторых сетях сервисы перевода могут быть недоступны.

**В: Как открыть PDF в Furigana Reader?**  
О: Несколько способов: «Открыть чтение PDF» во всплывающем окне, щелчок правой кнопкой по ссылке на PDF и «Открыть PDF в Furigana Reader», либо включите «Умную детекцию PDF» для автоматического перенаправления PDF-URL во встроенный просмотрщик.

**В: Умная детекция PDF включена, но часть PDF не перенаправляется**  
О: Автоперенаправление срабатывает для URL, оканчивающихся на `.pdf`. Для PDF без расширения `.pdf` в адресе или открытых во встроенном просмотре Chrome появится уведомление или значок с предложением открыть в Furigana Reader.

**В: Можно ли пользоваться словарём без интернета?**  
О: Да. Словарь JMdict (более 180 000 записей) полностью входит в состав расширения. Все запросы выполняются локально, без сети.
