# Furigana Reader - Användarguide

> Version: v1.3.0

## Introduktion

Furigana Reader är ett webbläsartillägg för dig som lär dig japanska. Med en WASM-morfologisk analysator (Lindera) och JavaScript-reserv lägger det till furigana (läsnoteringar) över kanji på webbsidor så att du enklare kan lära dig uttal.

---

## Huvudfunktioner

- **Annotering av markerad text** — Markera japansk text för att automatiskt visa furigana och tal-knappar
- **Furigana för hela sidan** — Lägg till furigana för alla kanji med ett klick
- **Text till tal** — Klicka på högtalarikonen för att höra uttal
- **Uppläsning av markering** — Markera text, klicka på den flytande knappen eller högerklicka «Läs upp markering»
- **Verktygstips vid hovring** — Hovra över annoterade tecken för furigana och uttalsknappar
- **Flera furigana-stilar** — Hiragana, Katakana och Romaji
- **Flerspråkigt gränssnitt** — 38 gränssnittsspråk

---

## Så använder du tillägget

### Steg 1: Installera tillägget

Installera **Furigana Reader** från [Chrome Web Store](https://chromewebstore.google.com/) eller ladda det lokalt i utvecklarläge.

### Steg 2: Öppna en webbsida

Besök en sida med japanskt innehåll.

### Steg 3: Markera text eller använd flytande knapp

Markera japansk text eller klicka på den flytande knappen nere till höger för furigana-läge för hela sidan.

### Steg 4: Visa furigana

Hovra över tecken för verktygstips; klicka på högtalaren för uttal.

### Steg 5: Läs upp markerad text

Markera japansk text med musen, klicka på den flytande högtalaren eller högerklicka «Läs upp markering».

> **Tips:** Klicka på tilläggsikonen i verktygsfältet för inställningar (furigana-stil, talhastighet m.m.).

## Inställningar

| Inställning | Beskrivning |
|-------------|-------------|
| **Aktivera furigana** | Huvudbrytare för furigana-funktionen |
| **Furigana för hela sidan** | Visar furigana för alla kanji (kan påverka layout) |
| **Furigana-stil** | Välj Hiragana, Katakana eller Romaji |
| **Talhastighet** | Justera hastighet för meningsuppläsning |
| **Verktygstips vid hovring** | Visa furigana-verktygstips när musen hovrar |

## Vanliga frågor

**F: Varför fungerar det inte på vissa sidor?**  
**S:** Av säkerhetsskäl kan tillägg inte köras på specialsidor som `chrome://`, webbläsarinställningar eller Chrome Web Store.

**F: Vad om furigana är felaktig?**  
**S:** Sällsynta ord eller särskilda läsningar (熟字訓) kan bli fel. Vi förbättrar kontinuerligt — dela gärna konkreta exempel.

**F: Inget ljud från text till tal?**  
**S:** Kontrollera systemvolymen och att japanska röster är installerade. Stöd varierar mellan webbläsare och OS.

**F: Påverkar hela sidan-läget layouten?**  
**S:** Furigana behöver extra utrymme. Stäng av läget och använd verktygstips om det stör.
