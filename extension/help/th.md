# Furigana Reader — คู่มือการใช้งาน

> เวอร์ชัน: v1.4.1

## บทนำ

Furigana Reader เป็นส่วนขยายเบราว์เซอร์สำหรับผู้เรียนภาษาญี่ปุ่น ขับเคลื่อนด้วยตัววิเคราะห์เชิงรูปศัพท์แบบ WASM (Lindera + IPAdic) พร้อมทางสำรองด้วย JavaScript ช่วยเพิ่มฟุริงานะ (การอ่านกำกับ) ให้กับตัวอักษรคันจิบนหน้าเว็บและใน PDF ได้อย่างแม่นยำ นอกจากนี้ยังมีพจนานุกรมภาษาญี่ปุ่นในตัว การอ่านข้อความเป็นคำพูด และการแปล — ช่วยให้ฝึกออกเสียงภาษาญี่ปุ่นได้ง่ายขึ้น

---

## คุณสมบัติหลัก

- **โหมดฟุริงานะทั้งหน้า** — คลิกครั้งเดียวเพื่อใส่ฟุริงานะให้คันจิทั้งหมดบนหน้า
- **พจนานุกรมเมื่อชี้** — ชี้ที่ตัวอักษรที่มีกำกับเพื่อดูคำจำกัดความจาก JMdict (มากกว่า 180,000 รายการ) การอ่าน ป้ายระดับ JLPT (N5–N1) และปุ่มออกเสียง เลือกโหมด พจนานุกรม / การอ่าน / ปิด
- **ตัวอ่าน PDF** — ตัวอ่าน PDF ในตัวพร้อมฟุริงานะ พจนานุกรม การพูด และการแปล รองรับลากวาง โหลดจาก URL และตรวจจับ PDF อัตโนมัติพร้อมเปลี่ยนเส้นทางอัจฉริยะ
- **สามรูปแบบการอ่าน** — ฮิระงะนะ คะตะกะนะ และโรมะจิ
- **แยกโอะคุริงะนะ** — แยกโอะคุริงะนะ (送り仮名) ออกจากรากคันจิได้ถูกต้อง
- **รองรับจูกุจิคุน** — การอ่านที่ถูกต้องสำหรับคำประกอบหลายคันจิที่มีการอ่านพิเศษ (熟字訓)
- **สังเคราะห์เสียง (TTS)** — คลิกปุ่มลำโพงเพื่อฟังการออกเสียงภาษาญี่ปุ่น
- **อ่านข้อความที่เลือกและคาราโอเกะ** — เลือกข้อความภาษาญี่ปุ่นใดก็ได้ แถบเครื่องมือกะทัดรัดจะปรากฏพร้อมปุ่มพูดและแปล เมื่ออ่านออกเสียงจะมีการไฮไลต์ทีละคำหรือทีละตัวอักษรแบบเรียลไทม์ (เอฟเฟกต์คาราโอเกะ) ซิงค์กับเสียง
- **แปลข้อความที่เลือก** — เลือกข้อความใดก็ได้ คลิกปุ่มแปลในแถบเครื่องมือเพื่อแปลทันทีผ่าน Bing Translate หรือ Google Translate แสดงในบับเบิลแบบอินไลน์
- **แป้นพิมพ์ลัด** — เข้าถึงคุณสมบัติหลักได้เร็วผ่านปุ่มลัดที่ปรับแต่งได้
- **หลายภาษาในอินเทอร์เฟซ** — รองรับ 38 ภาษาของอินเทอร์เฟซ

## วิธีใช้งาน

### ขั้นตอนที่ 1: ติดตั้งส่วนขยาย

ติดตั้ง **Furigana Reader** จาก [Chrome Web Store](https://chromewebstore.google.com/) หรือโหลดในโหมดนักพัฒนาแบบในเครื่อง

### ขั้นตอนที่ 2: เปิดหน้าเว็บใดก็ได้

เข้าหน้าที่มีเนื้อหาภาษาญี่ปุ่น

### ขั้นตอนที่ 3: เปิดใช้ฟุริงานะ

คลิกไอคอนส่วนขยายบนแถบเครื่องมือ เปิด «เปิดใช้ฟุริงานะ» จากนั้นเปิด «ฟุริงานะทั้งหน้า» เพื่อกำกับคันจิทั้งหน้า คุณยังใช้ปุ่มลอยมุมล่างขวาได้

### ขั้นตอนที่ 4: ดูฟุริงานะ

ชี้ที่ตัวอักษรเพื่อดูคำแนะนำฟุริงานะพร้อมการอ่านและคำจำกัดความจากพจนานุกรม คลิกไอคอนลำโพงเพื่อฟังการออกเสียง

### ขั้นตอนที่ 5: อ่านและแปลข้อความที่เลือก

เลือกข้อความภาษาญี่ปุ่นด้วยเมาส์ แถบเครื่องมือกะทัดรัดจะปรากฏใกล้การเลือกพร้อมสองปุ่ม:
- **🔊 พูด** — อ่านข้อความที่เลือกออกเสียงพร้อมไฮไลต์แบบคาราโอเกะ
- **🌐 แปล** — แสดงบับเบิลแปลแบบอินไลน์ใต้แถบเครื่องมือ

คุณยังคลิกขวาแล้วเลือก «Furigana Reader > อ่านข้อความที่เลือกออกเสียง» หรือ «Furigana Reader > แปลข้อความที่เลือก»

> **เคล็ดลับ:** คลิกไอคอนส่วนขยายเพื่อเปิดแผงการตั้งค่าและปรับสไตล์ฟุริงานะ โหมดเมื่อชี้ ความเร็วการพูด เครื่องมือแปล และอื่นๆ

## พจนานุกรมเมื่อชี้

ส่วนขยายมีพจนานุกรมภาษาญี่ปุ่นในตัวจาก JMdict (มากกว่า 180,000 รายการ) คุณเลือกโหมดเมื่อชี้ได้หลายแบบในการตั้งค่า:

| โหมด | พฤติกรรม |
|------|----------|
| **พจนานุกรม** | เมื่อชี้แสดงการอ่าน + คำจำกัดความ + ระดับ JLPT + ปุ่มออกเสียง |
| **การอ่าน** | เมื่อชี้แสดงการอ่าน + ปุ่มออกเสียง (ไม่มีคำจำกัดความ) |
| **ปิด** | ไม่มีเอฟเฟกต์เมื่อชี้ |

ในโหมด **พจนานุกรม** คำแนะนำแสดง:
- คำและการอ่าน (ฟุริงานะ)
- ปุ่มออกเสียง (คลิกเพื่อฟัง)
- คำจำกัดความภาษาญี่ปุ่นจาก JMdict
- ป้ายระดับ JLPT (N5–N1) เมื่อมี

> **เคล็ดลับ:** ข้อมูลพจนานุกรมโหลดตามต้องการเมื่อเปิดโหมดพจนานุกรม และถูกถอดเมื่อเปลี่ยนโหมดอื่นเพื่อประหยัดหน่วยความจำ

## ตัวอ่าน PDF

Furigana Reader มีตัวอ่าน PDF ในตัว ให้คุณอ่านเอกสาร PDF พร้อมฟุริงานะ พจนานุกรม การพูด และการแปล — คุณสมบัติเดียวกับบนหน้าเว็บ ใช้กับ PDF ได้

### การเปิด PDF

**วิธีที่ 1: จากป๊อปอัป**  
คลิกไอคอนส่วนขยาย แล้วคลิก «เปิดตัวอ่าน PDF» ลากวางไฟล์ PDF หรือคลิก «เลือกไฟล์» เพื่อเปิด PDF ในเครื่อง คุณยังวาง URL ของ PDF ได้

**วิธีที่ 2: เมนูบริบท**  
คลิกขวาลิงก์ `.pdf` บนหน้าเว็บ แล้วเลือก «เปิด PDF ด้วย Furigana Reader»

**วิธีที่ 3: ตรวจจับอัตโนมัติ**  
เมื่อเปิด «การตรวจจับ PDF อัจฉริยะ» ในการตั้งค่า ส่วนขยายจะเปลี่ยนเส้นทาง URL ที่ลงท้ายด้วย `.pdf` ไปยังตัวอ่านในตัว เมื่อตรวจพบ PDF แต่ไม่เปลี่ยนเส้นทาง (เช่น ตัวดูในตัวของ Chrome) คุณจะเห็นการแจ้งเตือนและคำเชิญให้เปิดใน Furigana Reader

### คุณสมบัติตัวอ่าน PDF

- **กำกับฟุริงานะ** — คุณสมบัติฟุริงานะทั้งหมดใช้กับข้อความใน PDF รวมโหมดทั้งหน้าและคำแนะนำเมื่อชี้
- **ห้าโหมดฟุริงานะ** — ฮิระงะนะ คะตะกะนะ โรมะจิ เฉพาะเมื่อชี้ และปิด
- **พจนานุกรมคลิก** — คลิกคำใดก็ได้เพื่อดูคำจำกัดความจาก JMdict (ใน PDF ใช้คลิกแทนการชี้เพื่ออ่านโฟกัส)
- **แถบเครื่องมือการเลือก** — เลือกข้อความเพื่อพูด แปล หรือคัดลอก
- **แถบด้านข้าง** — สารบัญและภาพย่อหน้า
- **ค้นหา** — ค้นหาข้อความเต็มใน PDF
- **ธีม** — ธีมอ่านมืด สว่าง และเซเปีย
- **ซูม** — หลายระดับซูม รวมฟุริงานะปรับตามซูม
- **แป้นพิมพ์ลัด** — ลูกศรนำทาง +/- ซูม Ctrl/Cmd+F ค้นหา

## อ่านข้อความที่เลือกและคาราโอเกะ

การอ่านข้อความที่เลือกให้เลือกข้อความภาษาญี่ปุ่นใดก็ได้แล้วอ่านออกเสียงคลิกเดียว — เหมาะสำหรับฝึกออกเสียงประโยคและการอ่าน

**วิธีที่ 1: แถบเครื่องมือการเลือก**  
เลือกข้อความภาษาญี่ปุ่นด้วยเมาส์ แถบกะทัดรัดปรากฏใกล้การเลือกพร้อมปุ่ม 🔊 พูด และ 🌐 แปล คลิกพูด ขณะอ่าน แต่ละคำหรือตัวอักษรจะถูกไฮไลต์แบบเรียลไทม์ (คาราโอเกะ) เพื่อติดตาม

**วิธีที่ 2: เมนูคลิกขวา**  
หลังเลือกข้อความภาษาญี่ปุ่น คลิกขวาแล้วเลือก «Furigana Reader > อ่านข้อความที่เลือกออกเสียง»

**วิธีที่ 3: แป้นพิมพ์ลัด**  
เลือกข้อความแล้วกด `Alt+Shift+S` (Mac: `Ctrl+Shift+S`)

> **เคล็ดลับ:** เอฟเฟกต์ไฮไลต์คาราโอเกะทำงานได้ดีที่สุดเมื่อเบราว์เซอร์รองรับเหตุการณ์ขอบเขตคำของ TTS หากไม่รองรับ ส่วนขยายจะใช้ทางสำรองตามเวลาเพื่อไฮไลต์ลื่น

## การแปล

เลือกข้อความใดก็ได้บนหน้าแล้วใช้การแปลเพื่อรับคำแปลทันที

**วิธีที่ 1: แถบเครื่องมือการเลือก**  
เลือกข้อความ แล้วคลิกปุ่ม 🌐 แปลในแถบเครื่องมือ บับเบิลแปลปรากฏด้านล่างพร้อมผลลัพธ์และปุ่มคัดลอก

**วิธีที่ 2: เมนูคลิกขวา**  
เลือกข้อความ คลิกขวาแล้วเลือก «Furigana Reader > แปลข้อความที่เลือก»

**วิธีที่ 3: แป้นพิมพ์ลัด**  
เลือกข้อความแล้วกด `Alt+Shift+T` (Mac: `Ctrl+Shift+T`)

**เครื่องมือแปล:**
- **Bing Translate** (ค่าเริ่มต้น) — Microsoft Translator
- **Google Translate** — Google

ทั้งสองรองรับ **108 ภาษาเป้าหมาย**

สลับเครื่องมือแปลและภาษาเป้าหมายในการตั้งค่าส่วนขยาย ภาษาเป้าหมายตรวจจับอัตโนมัติจากภาษาเบราว์เซอร์

> **เคล็ดลับ:** คลิกนอกแถบเครื่องมือหรือบับเบิลเพื่อปิด

## แป้นพิมพ์ลัด

| ปุ่มลัด | ปุ่มลัด Mac | การทำงาน |
|---------|--------------|----------|
| `Alt+Shift+F` | `Ctrl+Shift+F` | เปิด/ปิดกำกับฟุริงานะ |
| `Alt+Shift+S` | `Ctrl+Shift+S` | อ่านข้อความที่เลือกออกเสียง |
| `Alt+Shift+T` | `Ctrl+Shift+T` | แปลข้อความที่เลือก |

> **เคล็ดลับ:** ปรับแต่งปุ่มลัดใน Chrome ที่ `chrome://extensions/shortcuts`

## คู่มือการตั้งค่า

| การตั้งค่า | คำอธิบาย |
|------------|----------|
| **เปิดใช้ฟุริงานะ** | สวิตช์หลักเปิด/ปิดคุณสมบัติกำกับฟุริงานะ |
| **ฟุริงานะทั้งหน้า** | เมื่อเปิด แสดงฟุริงานะให้คันจิทั้งหมด (อาจกระทบเลย์เอาต์หน้า) |
| **โหมดเมื่อชี้** | เลือกพฤติกรรมเมื่อชี้: พจนานุกรม (การอ่าน + คำจำกัดความ + JLPT + เสียง) การอ่าน (การอ่าน + เสียง) หรือปิด |
| **สไตล์ฟุริงานะ** | ฮิระงะนะ คะตะกะนะ หรือโรมะจิ |
| **ความเร็วการพูด** | ปรับความเร็วการอ่านประโยค |
| **เครื่องมือแปล** | เลือก Bing Translate หรือ Google Translate |
| **ภาษาเป้าหมาย** | ตั้งภาษาแปลเป้าหมาย (ตรวจจับอัตโนมัติจากภาษาเบราว์เซอร์) |
| **การตรวจจับ PDF อัจฉริยะ** | เมื่อเปิด เปลี่ยนเส้นทาง URL PDF ไปตัวอ่านในตัวโดยอัตโนมัติและแจ้งเตือนเมื่อตรวจพบ PDF |

## คำถามที่พบบ่อย

**ถ: ทำไมบางหน้าใช้ไม่ได้?**  
ต: เพื่อความปลอดภัย ส่วนขยายไม่ทำงานบนหน้าพิเศษ เช่น `chrome://` การตั้งค่าเบราว์เซอร์ หรือ Chrome Web Store

**ถ: ฟุริงานะไม่ตรงทำอย่างไร?**  
ต: คำหายากหรือการอ่านพิเศษ (熟字訓) อาจผิดพลาด เราพัฒนาอย่างต่อเนื่อง กรุณาแชร์ตัวอย่างเพื่อช่วยปรับปรุง

**ถ: TTS ไม่มีเสียง?**  
ต: ตรวจสอบระดับเสียงระบบและแพ็กเสียงภาษาญี่ปุ่น การรองรับการพูดแตกต่างตามเบราว์เซอร์และระบบปฏิบัติการ

**ถ: โหมดทั้งหน้ากระทบเลย์เอาต์?**  
ต: ฟุริงานะต้องการพื้นที่เพิ่ม อาจกระทบเลย์เอาต์เดิม หากอ่านยาก ปิดโหมดทั้งหน้าแล้วใช้คำแนะนำเมื่อชี้แทน

**ถ: แปลไม่ทำงาน?**  
ต: การแปลต้องใช้อินเทอร์เน็ต หาก Bing Translate ล้มเหลว ลองสลับเป็น Google Translate ในบางเครือข่ายอาจบล็อกบริการแปล

**ถ: เปิด PDF ด้วย Furigana Reader อย่างไร?**  
ต: คลิก «เปิดตัวอ่าน PDF» ในป๊อปอัป คลิกขวาลิงก์ PDF แล้วเลือก «เปิด PDF ด้วย Furigana Reader» หรือเปิด «การตรวจจับ PDF อัจฉริยะ» เพื่อเปลี่ยนเส้นทางอัตโนมัติ

**ถ: เปิดการตรวจจับ PDF อัจฉริยะแล้วแต่บาง PDF ไม่เปลี่ยนเส้นทาง?**  
ต: การเปลี่ยนอัตโนมัติใช้กับ URL ที่ลงท้าย `.pdf` สำหรับ PDF ที่ไม่มีนามสกุลหรือเปิดในตัวดูของ Chrome จะมีการแจ้งเตือนและป้ายเชิญให้เปิดใน Furigana Reader

**ถ: ใช้พจนานุกรมออฟไลน์ได้ไหม?**  
ต: ได้ พจนานุกรม JMdict (มากกว่า 180,000 รายการ) รวมอยู่ในส่วนขยายทั้งหมด การค้นหาทำในเครื่อง ไม่ต้องใช้เครือข่าย
