# Furigana Reader - 使用者指南

> 版本：v1.4.1

## 簡介

Furigana Reader 是為日語學習者設計的瀏覽器擴充功能。透過 WASM 形態素分析器（Lindera + IPAdic）並具 JavaScript 後援，可為網頁與 PDF 上的漢字準確加上讀音（振假名／注音）標註。另內建日語辭典、文字轉語音與翻譯功能，協助您更輕鬆學習日語發音。

---

## 主要功能

- **整頁振假名模式** — 一鍵為頁面上所有漢字加上讀音標註
- **懸浮辭典** — 將游標移到已標註字元上可查看 JMdict 釋義（逾 18 萬筆）、讀音、JLPT 等級徽章（N5–N1）與發音按鈕；可選辭典模式、讀音模式或關閉
- **PDF 閱讀器** — 內建 PDF 閱讀器，支援振假名、辭典、朗讀與翻譯；支援拖放、URL 載入，以及自動偵測 PDF 並智慧重新導向
- **三種讀音樣式** — 支援平假名、片假名與羅馬字顯示
- **送假名拆分** — 準確將送假名（送り仮名）與漢字詞幹分開
- **熟字訓支援** — 正確處理多字漢字複合詞的特殊讀法（熟字訓）
- **文字轉語音** — 點選喇叭按鈕聆聽日語發音
- **選取朗讀與卡拉 OK 效果** — 選取任意日語文字後會出現精簡工具列，含朗讀與翻譯按鈕；朗讀時會依音訊同步即時逐字或逐詞高亮（卡拉 OK 效果）
- **選取翻譯** — 選取任意文字後點工具列上的翻譯按鈕，透過 Bing 或 Google 翻譯取得即時翻譯，以內嵌氣泡顯示
- **鍵盤快速鍵** — 可自訂快速鍵，快速使用核心功能
- **多語介面** — 支援 38 種介面語言

## 使用方式

### 步驟一：安裝擴充功能
從 [Chrome 線上應用程式商店](https://chromewebstore.google.com/)安裝 **Furigana Reader**，或以開發人員模式本機載入。

### 步驟二：開啟網頁
造訪任何含日語內容的網頁。

### 步驟三：啟用振假名
點選瀏覽器工具列上的擴充功能圖示。開啟「啟用振假名」，再開啟「整頁振假名」以為頁面上所有漢字加上標註。您也可點選右下角的浮動按鈕。

### 步驟四：檢視振假名
將游標移到字元上可看到含讀音與辭典釋義的振假名工具提示。點選喇叭圖示可聽發音。

### 步驟五：朗讀與翻譯選取文字
以滑鼠選取任意日語文字。選取範圍附近會出現精簡工具列，上有兩個按鈕：
- **🔊 朗讀** — 以卡拉 OK 式高亮朗讀選取文字
- **🌐 翻譯** — 在工具列下方顯示內嵌翻譯氣泡

您亦可右鍵選擇「Furigana Reader > 朗讀選取內容」或「Furigana Reader > 翻譯選取內容」。

> **提示：** 點選瀏覽器工具列上的擴充功能圖示可開啟設定面板，調整振假名樣式、懸浮模式、語速、翻譯引擎等。

## 懸浮辭典

擴充功能內建以 JMdict（逾 18 萬筆）為基礎的日語辭典。您可在設定中選擇多種懸浮模式：

| 模式 | 行為 |
|------|------|
| **辭典** | 懸浮時顯示讀音 + 釋義 + JLPT 等級 + 發音按鈕 |
| **讀音** | 懸浮時僅顯示讀音 + 發音按鈕（無釋義） |
| **關閉** | 無懸浮效果 |

在 **辭典模式**下，工具提示會顯示：
- 詞彙與讀音（振假名）
- 發音按鈕（點選播放）
- 來自 JMdict 的日語釋義
- 若有資料則顯示 JLPT 等級徽章（N5–N1）

> **提示：** 辭典資料在啟用辭典模式時才會依需求載入，切換至其他模式時會卸載以節省記憶體。

## PDF Reader

Furigana Reader 內建 PDF 閱讀器，讓您在 PDF 上也能使用與網頁相同的振假名、辭典、朗讀與翻譯功能。

### 開啟 PDF

**方式一：從彈出視窗**  
點選擴充功能圖示，再點「開啟 PDF 閱讀器」。拖放 PDF 檔或點「選擇檔案」開啟本機 PDF。您也可貼上 PDF 網址。

**方式二：情境選單**  
在網頁上對 `.pdf` 連結右鍵，選擇「以 Furigana Reader 開啟 PDF」。

**方式三：自動偵測**  
在設定中啟用「PDF 智慧偵測」時，擴充功能會自動將 `.pdf` 網址重新導向至內建閱讀器。若偵測到 PDF 但未重新導向（例如 Chrome 內建檢視器），您會看到通知與提示，引導您在 Furigana Reader 中開啟。

### PDF 閱讀器功能

- **振假名標註** — 整頁模式與懸浮工具提示等所有振假名功能皆適用於 PDF 文字
- **五種振假名模式** — 平假名、片假名、羅馬字、僅懸浮與關閉
- **點擊辭典** — 點選任意詞彙可查看 JMdict 釋義（PDF 為減少干擾，以點擊取代懸浮）
- **選取工具列** — 選取文字以朗讀、翻譯或複製
- **側邊欄** — 目錄大綱與頁面縮圖
- **搜尋** — PDF 全文搜尋
- **主題** — 深色、淺色與仿舊紙色閱讀主題
- **縮放** — 多種縮放比例，含隨縮放調整的振假名
- **鍵盤快速鍵** — 方向鍵導覽、+/- 縮放、Ctrl/Cmd+F 搜尋

## 選取朗讀與卡拉 OK

選取朗讀功能可讓您選取任意日語文字並一鍵朗讀，適合練習句型發音與閱讀。

**方式一：選取工具列**  
以滑鼠選取日語文字。選取範圍附近會出現含 🔊 朗讀與 🌐 翻譯按鈕的精簡工具列。點朗讀按鈕開始播放。朗讀時會即時高亮每個字或詞（卡拉 OK 效果），方便跟讀。

**方式二：右鍵選單**  
選取日語文字後右鍵，選擇「Furigana Reader > 朗讀選取內容」。

**方式三：鍵盤快速鍵**  
選取文字後按下 `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）。

> **提示：** 卡拉 OK 高亮在瀏覽器支援 TTS 字詞邊界事件時效果最佳。若不支援，擴充功能會以時間為基礎的後援方式維持平滑高亮。

---

## 翻譯

在頁面上選取文字後，可使用翻譯功能取得即時翻譯。

**方式一：選取工具列**  
選取文字後點工具列上的 🌐 翻譯按鈕。下方會出現翻譯氣泡，顯示結果與複製按鈕。

**方式二：右鍵選單**  
選取文字後右鍵，選擇「Furigana Reader > 翻譯選取內容」。

**方式三：鍵盤快速鍵**  
選取文字後按下 `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）。

**翻譯引擎：**
- **Bing 翻譯**（預設）— 由 Microsoft Translator 提供
- **Google 翻譯** — 由 Google 提供

兩者皆支援 **108 種目標語言**。

您可在擴充功能設定中切換翻譯引擎與目標語言。目標語言會依瀏覽器語言自動偵測。

> **提示：** 在工具列或氣泡外任意處點一下即可關閉。

## 鍵盤快速鍵

| 快速鍵 | Mac 快速鍵 | 動作 |
|--------|------------|------|
| `Alt+Shift+F` | `Ctrl+Shift+F` | 開關振假名標註 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 朗讀選取文字 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 翻譯選取文字 |

> **提示：** 可在 Chrome 的 `chrome://extensions/shortcuts` 自訂這些快速鍵。

## 設定說明

| 設定 | 說明 |
|------|------|
| **啟用振假名** | 開啟或關閉振假名標註功能的主開關 |
| **整頁振假名** | 啟用時為所有漢字顯示振假名（可能影響版面） |
| **懸浮模式** | 選擇懸浮行為：辭典（讀音 + 釋義 + JLPT + 音訊）、讀音（讀音 + 音訊）或關閉 |
| **振假名樣式** | 選擇平假名、片假名或羅馬字顯示 |
| **語速** | 調整句子朗讀速度 |
| **翻譯引擎** | 選擇 Bing 翻譯或 Google 翻譯 |
| **目標語言** | 設定翻譯目標語言（依瀏覽器語言自動偵測） |
| **PDF 智慧偵測** | 啟用時自動將 PDF 網址重新導向內建閱讀器，並在偵測到 PDF 時顯示通知 |

## FAQ

**問：為什麼在某些頁面上無法使用？**  
答：基於安全考量，擴充功能無法在 `chrome://`、瀏覽器設定或 Chrome 線上應用程式商店等特殊頁面上執行。

**問：振假名不準怎麼辦？**  
答：少見詞彙或特殊讀法（熟字訓）可能仍有誤差。我們持續改進，歡迎提供具體案例協助我們改善。

**問：文字轉語音沒有聲音？**  
答：請檢查系統音量並確認已安裝日語語音套件。語音支援因瀏覽器與作業系統而異。

**問：整頁模式會影響版面？**  
答：振假名標註需要額外空間，可能影響原始版面。若影響閱讀，請關閉整頁模式，改只用懸浮工具提示。

**問：翻譯無法使用？**  
答：翻譯需要網路連線。若 Bing 翻譯失敗，請在設定中改試 Google 翻譯。部分網路環境可能封鎖翻譯服務。

**問：如何用 Furigana Reader 開啟 PDF？**  
答：可在彈出視窗點「開啟 PDF 閱讀器」、對 PDF 連結右鍵選「以 Furigana Reader 開啟 PDF」，或在設定中啟用「PDF 智慧偵測」以自動將 `.pdf` 網址導向內建閱讀器。

**問：已啟用 PDF 智慧偵測，但部分 PDF 沒有重新導向？**  
答：自動重新導向適用於以 `.pdf` 結尾的網址。若 PDF 沒有 `.pdf` 副檔名或由 Chrome 內建檢視器開啟，您會看到通知與徽章，提示您在 Furigana Reader 中開啟。

**問：可以離線使用辭典嗎？**  
答：可以。JMdict 辭典（逾 18 萬筆）完整內建於擴充功能中，所有查詢皆在本地完成，無需網路。
