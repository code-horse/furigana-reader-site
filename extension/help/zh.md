# Furigana Reader - 用户指南

> 版本：v1.4.0

## 简介

Furigana Reader 是一款面向日语学习者的浏览器扩展。它采用 WASM 形态素分析引擎（Lindera + IPAdic），并可在需要时回退到 JavaScript，能够准确为网页和 PDF 上的汉字添加注音假名（读音标注）。同时内置日语词典、文本转语音和翻译功能，帮助您更轻松地学习日语发音。

---

## 主要功能

- **整页注音模式** — 一键为页面上所有汉字添加注音假名
- **悬停词典** — 将鼠标悬停在已标注字符上，可查看 JMdict 释义（18万+词条）、读音、JLPT 等级标签（N5–N1）及发音按钮；可选择词典模式、读音模式或关闭
- **PDF 阅读器** — 内置 PDF 阅读器，支持注音假名、词典、朗读和翻译功能；支持拖放、URL 加载和 PDF 智能检测自动跳转
- **多种注音样式** — 支持平假名、片假名与罗马字显示
- **送り仮名分割** — 准确分离送り假名与汉字词干
- **熟字训支持** — 正确处理多汉字词组的特殊读音（熟字訓）
- **文本转语音（TTS）** — 点击扬声器按钮收听日语读音
- **选区朗读 + 卡拉OK效果** — 选中任意日文文本，弹出紧凑工具栏（朗读 + 翻译按钮），朗读时逐词或逐字高亮同步显示（卡拉OK效果）
- **选区翻译** — 选中文本后点击工具栏翻译按钮，通过 Bing 或 Google 翻译即时获取翻译结果，以悬浮气泡显示
- **快捷键** — 通过可自定义的键盘快捷键快速使用核心功能
- **多语言界面** — 支持 38 种界面语言

## 使用方法

### 第 1 步：安装扩展

从 [Chrome 网上应用店](https://chromewebstore.google.com/) 安装 **Furigana Reader**，或在开发者模式下本地加载。

### 第 2 步：打开任意网页

访问包含日文内容的任意网页。

### 第 3 步：启用注音假名

点击浏览器工具栏中的扩展图标，开启「启用注音假名」，然后开启「整页注音假名」为页面所有汉字添加注音。也可以点击右下角的浮动按钮。

### 第 4 步：查看注音假名

将鼠标悬停在字符上可查看注音提示和词典释义，点击扬声器图标可收听读音。

### 第 5 步：朗读与翻译所选文本

用鼠标选中任意日文文本，选区附近会出现一个紧凑工具栏，包含两个按钮：
- **🔊 朗读** — 朗读选中文本，同时逐词或逐字高亮显示（卡拉OK效果）
- **🌐 翻译** — 在工具栏下方显示翻译结果气泡

也可以右键选择「Furigana Reader > 朗读所选内容」或「Furigana Reader > 翻译选区」。

> **提示：** 点击浏览器工具栏中的扩展图标可打开设置面板，调整注音样式、悬停模式、语速、翻译引擎等。

## 悬停词典

扩展内置日语词典，基于 JMdict（18万+词条）。您可以在设置中选择悬停模式：

| 模式 | 行为 |
|------|------|
| **词典模式** | 悬停显示读音 + 释义 + JLPT 等级 + 发音按钮 |
| **读音模式** | 悬停显示读音 + 发音按钮（无释义） |
| **关闭** | 无悬停效果 |

在**词典模式**下，提示框显示：
- 词语及其读音（注音假名）
- 发音按钮（点击收听）
- JMdict 日语释义
- JLPT 等级标签（N5–N1，如有）

> **提示：** 词典数据在启用词典模式时按需加载，切换到其他模式时自动卸载以节省内存。

## PDF 阅读器

Furigana Reader 内置 PDF 阅读器，让您在阅读 PDF 文档时也能使用注音假名、词典、朗读和翻译功能——网页上的所有功能，现在 PDF 也能用。

### 打开 PDF

**方式一：从弹出窗口打开**  
点击扩展图标，然后点击「打开 PDF 阅读器」。拖放 PDF 文件或点击「选择文件」打开本地 PDF，也可以粘贴 PDF 网址。

**方式二：右键菜单**  
在网页上右键点击任意 `.pdf` 链接，选择「用 Furigana Reader 打开 PDF」。

**方式三：自动检测**  
在设置中启用「PDF 智能检测」后，扩展会自动将 `.pdf` URL 跳转到内置阅读器。当检测到 PDF 但无法自动跳转时（如 Chrome 内置查看器），会显示通知提示您在 Furigana Reader 中打开。

### PDF 阅读器功能

- **注音假名标注** — 所有注音功能均可在 PDF 文本上使用，包括整页模式和悬停提示
- **五种注音模式** — 平假名、片假名、罗马音、仅悬浮、隐藏
- **点击查词典** — 点击任意词语查看 JMdict 释义（PDF 中使用点击而非悬停，提供更沉浸的阅读体验）
- **选区工具栏** — 选中文本可朗读、翻译或复制
- **侧边栏** — 目录大纲和页面缩略图
- **搜索** — PDF 全文搜索
- **主题** — 深色、浅色和护眼三种阅读主题
- **缩放** — 多种缩放级别，注音自适应缩放
- **快捷键** — 方向键翻页、+/- 缩放、Ctrl/Cmd+F 搜索

## 选区朗读与卡拉OK

选区朗读功能让您选中任意日文文本后一键朗读，非常适合学习句子发音和阅读练习。

**方式一：选区工具栏**  
用鼠标选中日文文本后，选区附近会出现一个紧凑工具栏，包含🔊朗读按钮和🌐翻译按钮。点击朗读按钮播放，播放时每个词或字会实时高亮显示（卡拉OK效果），帮助您跟读。

**方式二：右键菜单**  
选中日文文本后，右键选择「Furigana Reader > 朗读所选内容」。

**方式三：快捷键**  
选中文本后按 `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）即可朗读。

> **提示：** 卡拉OK高亮效果在浏览器支持 TTS 词边界事件时效果最佳。若不支持，扩展会使用基于计时的备用方案实现平滑高亮。

## 翻译

选中页面上的任意文本，使用翻译功能即可获取即时翻译。

**方式一：选区工具栏**  
选中文本后，点击工具栏中的🌐翻译按钮，下方会弹出翻译结果气泡，带有复制按钮。

**方式二：右键菜单**  
选中文本后，右键选择「Furigana Reader > 翻译选区」。

**方式三：快捷键**  
选中文本后按 `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）即可翻译。

**翻译引擎：**
- **Bing 翻译**（默认）— 由微软翻译提供支持
- **Google 翻译** — 由 Google 提供支持

两种引擎均支持 **108 种目标语言**。

您可以在扩展设置中切换翻译引擎和目标语言。目标语言会根据您的浏览器语言自动检测。

> **提示：** 点击工具栏或气泡外的任意位置即可关闭。

|---------|-------------|

| **整页注音假名** | 开启后为所有汉字显示注音（可能影响页面排版） |

| **PDF 智能检测** | 开启后自动将 PDF URL 跳转到内置阅读器，并在检测到 PDF 时显示通知 |

## 常见问题

**Q：为什么在某些页面上无法使用？**  
A：出于安全原因，浏览器扩展无法在 `chrome://`、浏览器设置或 Chrome 网上应用店等特殊页面上运行。

**Q：注音不准确怎么办？**  
A：部分生僻词或特殊读音（如熟字训）可能出现错误。我们会持续改进，欢迎您提供具体例子。

**Q：文本转语音没有声音？**  
A：请检查系统音量，并确认已安装日语语音包。不同浏览器与操作系统的语音支持情况不同。

**Q：整页模式会影响排版吗？**  
A：注音假名需要额外空间，可能影响原页面布局。若影响阅读，请关闭整页模式，改用悬停提示。

**Q：翻译不工作？**  
A：翻译功能需要网络连接。如果 Bing 翻译失败，请尝试在设置中切换为 Google 翻译。某些网络环境可能会阻止访问翻译服务。

**Q：如何用 Furigana Reader 打开 PDF？**  
A：有多种方式：在弹出窗口中点击「打开 PDF 阅读器」，右键点击 PDF 链接选择「用 Furigana Reader 打开 PDF」，或在设置中启用「PDF 智能检测」自动跳转 PDF URL。

**Q：已启用 PDF 智能检测但某些 PDF 没有跳转？**  
A：自动跳转适用于以 `.pdf` 结尾的 URL。对于没有 `.pdf` 扩展名的 PDF 或在 Chrome 内置查看器中查看的 PDF，会显示通知和标记提示您在 Furigana Reader 中打开。

**Q：词典可以离线使用吗？**  
A：可以。JMdict 词典（18万+词条）完全内置于扩展中，所有查询均在本地完成，无需网络连接。
