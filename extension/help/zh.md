# Furigana Reader - 用户指南

> 版本：v2.0.0

## 简介

Furigana Reader 是一款面向日语学习者的浏览器扩展。它采用 WASM 形态素分析引擎（Lindera），并可在需要时回退到 JavaScript，能够准确为网页上的汉字添加注音假名（读音标注），帮助您更轻松地学习日语发音。

---

## 主要功能

- **文本选区注音** — 在网页上选择日文文本，自动显示注音假名与朗读按钮
- **整页注音模式** — 一键为页面上所有汉字添加注音假名
- **文本转语音（TTS）** — 点击扬声器按钮收听日语读音
- **选区朗读** — 选择任意日文文本，点击浮动按钮或右键选择「朗读所选内容」
- **悬停提示** — 将鼠标悬停在已标注字符上可查看注音假名与发音按钮
- **多种注音样式** — 支持平假名、片假名与罗马字显示
- **多语言界面** — 支持 38 种界面语言

## 使用方法

### 第 1 步：安装扩展

从 [Chrome 网上应用店](https://chromewebstore.google.com/) 安装 **Furigana Reader**，或在开发者模式下本地加载。

### 第 2 步：打开任意网页

访问包含日文内容的任意网页。

### 第 3 步：选择文本或使用浮动按钮

选择需要标注的日文文本，或点击右下角的浮动按钮以启用整页注音模式。

### 第 4 步：查看注音假名

将鼠标悬停在字符上可查看注音提示，点击扬声器图标可收听读音。

### 第 5 步：朗读所选文本

用鼠标选择任意日文文本，点击浮动扬声器按钮朗读；或右键选择「朗读所选内容」。

> **提示：** 点击浏览器工具栏中的扩展图标可打开设置面板，调整注音样式、语速等。

## 设置说明

| 设置 | 说明 |
|---------|-------------|
| **启用注音假名** | 总开关，用于开启或关闭注音功能 |
| **整页注音假名** | 开启后为所有汉字显示注音（可能影响页面排版） |
| **注音样式** | 在平假名、片假名或罗马字之间选择 |
| **语速** | 调整句子朗读速度 |
| **悬停提示** | 鼠标悬停时显示注音假名提示框 |

## 常见问题

**Q：为什么在某些页面上无法使用？**  
A：出于安全原因，浏览器扩展无法在 `chrome://`、浏览器设置或 Chrome 网上应用店等特殊页面上运行。

**Q：注音不准确怎么办？**  
A：部分生僻词或特殊读音（如熟字训）可能出现错误。我们会持续改进，欢迎您提供具体例子。

**Q：文本转语音没有声音？**  
A：请检查系统音量，并确认已安装日语语音包。不同浏览器与操作系统的语音支持情况不同。

**Q：整页模式会影响排版吗？**  
A：注音假名需要额外空间，可能影响原页面布局。若影响阅读，请关闭整页模式，改用悬停提示。
